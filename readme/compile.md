## Компиляция .html

Необходимо:
* `make`
* `php`

Выполните `make html`

Файл-результат: `c-book.html`

## Компиляция .pdf

Необходимо (в дополнение к `.html`):
* `pandoc`
* `pdflatex` (его установка вместе с необходимыми пакетами может быть нетривиальна)

Выполните `make pdf`

Файл-результат: `c-book.pdf`
